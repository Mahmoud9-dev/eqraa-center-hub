name: Frontend Responsive
description: Build responsive layouts that work beautifully across mobile, tablet, and desktop screen sizes. Use this skill when implementing layouts, handling breakpoints, or optimizing for different devices. Apply when working on layout components, grid systems, navigation that adapts to screen size, or any responsive design implementation. Essential for mobile-first development approach, using standard breakpoints consistently (sm, md, lg, xl), implementing fluid layouts with percentage widths, using rem/em units instead of pixels, ensuring touch-friendly tap targets (44x44px minimum), optimizing images for mobile, and prioritizing content visibility on smaller screens.
---

## When to use this skill:

- When implementing mobile-first layouts
- When adding Tailwind responsive modifiers (sm:, md:, lg:, xl:)
- When building navigation that adapts to mobile/desktop
- When creating grid layouts that reflow on smaller screens
- When setting font sizes that scale across breakpoints
- When implementing touch-friendly tap targets for mobile
- When optimizing images and assets for mobile performance
- When testing UI across different screen sizes
- When deciding content priority for mobile views
- When using percentage widths and relative units (rem, em)
- When implementing collapsible or hidden elements for mobile
